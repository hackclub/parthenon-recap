# Parthenon

Parthenon was a 3-day hackathon from November 14th to 16th as part of the Athena Initiative. Attendees qualified through the Athena Award, spending 30 hours building 3 projects before flying out to New York City. 

This website serves as a recap for the event. The old website can still be viewed at [old.parthenon.hackclub.com](https://old.parthenon.hackclub.com)

## Development

1. Clone the repository

    ```
    git clone https://github.com/hackclub/parthenon-recap && cd parthenon-recap
    ```

2. Install dependencies and start the development server
    ```
    bun install && bun run dev
    ```

3. Build the production build.
    ```
    bun run build
    ```
